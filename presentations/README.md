# Presentations

This folder contains buyer-facing presentation material for PHI Guard Intelligence.

## Files

- [PHI Guard Intelligence buyer deck](phi-guard-intelligence-buyer-deck.md): GitHub-friendly Markdown deck with visual assets.
- [Standalone HTML deck](phi-guard-intelligence-buyer-deck.html): Browser-ready presentation version.

## How To Present

Use the Markdown deck directly on GitHub for quick review. For a more polished meeting format, open the HTML deck in a browser and use print-to-PDF if a shareable PDF is needed.

The visuals are stored in `docs/assets/` as SVG files so they render on GitHub without extra dependencies.
