# Business And Compliance Readiness

This is the non-code checklist that must mature before hosted PHI processing.

## Agreements

```text
Business Associate Agreement template
cloud provider BAA
subprocessor inventory
vendor security review process
customer security packet
```

## Security Program

```text
security risk analysis
risk management plan
asset inventory
access review process
incident response plan
breach notification workflow
vulnerability management
penetration testing
secure SDLC
dependency and container scanning
backup and disaster recovery testing
```

## Operational Policies

```text
data retention and deletion policy
audit logging policy
encryption and key management policy
access control policy
acceptable use policy
secure support policy
change management policy
```

## Product Guardrails

```text
no raw PHI in telemetry
no raw PHI in logs
no raw PHI in LLM prompts
no raw PHI in error messages
raw upload deletion within configured TTL
human review for all findings
clear non-legal disclaimer
```

## Incident Workflow

```text
detect
triage
contain
preserve evidence
notify customer if required
support covered entity breach assessment
remediate
post-incident review
update controls
```
